# `rux fmt`

Format all `.rux` source files and the `Rux.toml` manifest.

```bash
rux fmt [--check] [--manifest]
```

## Options

| Flag         | Description                              |
| ------------ | ---------------------------------------- |
| `--check`    | Check formatting without modifying files |
| `--manifest` | Format only the manifest (`Rux.toml`)    |
