# `rux version`

Show detailed information about the Rux toolchain version.

```bash
rux version
```

Show brief version information.

```bash
rux -v
rux --version
```

## Examples

```bash
rux vesion
Name:     Rux
Version:  0.1.0
Commit:   a17d00a
Branch:   dev
Built:    2025-11-10 09:45:08
Profile:  Debug
Compiler: MSVC 1950
Target:   Windows x64
```

```bash
rux -v
Rux 0.1.0
```

```bash
rux --version
Rux 0.1.0
```
