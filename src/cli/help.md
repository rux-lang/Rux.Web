# `rux help`

Show help information for the CLI.

```bash
rux help
rux -h
rux --help
```

Show help information for a specific command.

```bash
rux help <command>
```

## Options

| Flag                          | Description            |
| ----------------------------- | ---------------------- |
| `--color [auto \| on \| off]` | Control colored output |

## Examples

```bash
rux help
...
```

```bash
rux help build
...
```
